🤖 Penerjemah AI

📚 Contoh Terjemahan

"Terserah kamu deh..."
→ "Saya merasa kesal karena pendapat saya tidak didengar. Saya ingin Anda mempertimbangkan perasaan saya dalam mengambil keputusan ini."
"Gak apa-apa kok"
→ "Sebenarnya saya merasa kecewa, tapi saya tidak ingin membuat masalah. Saya berharap Anda bisa memahami perasaan saya tanpa saya harus menjelaskan secara detail."
"Kamu pasti lupa ya..."
→ "Saya merasa tidak diprioritaskan. Hal ini penting bagi saya dan saya berharap Anda bisa lebih memperhatikan hal-hal yang berarti untuk saya."
"Udah tau aja deh..."
→ "Saya merasa Anda kurang peka terhadap situasi ini. Saya berharap Anda bisa lebih proaktif dalam memahami apa yang saya butuhkan."
"Capek deh ngomong sama kamu"
→ "Saya merasa frustrasi karena komunikasi kita tidak efektif. Saya butuh Anda mendengarkan dengan lebih baik dan memahami perspektif saya."
"Iya iya... bener kamu"
→ "Saya tidak setuju dengan Anda, tapi saya memilih untuk tidak berdebat. Saya berharap Anda bisa lebih terbuka terhadap perspektif yang berbeda."
